# Aljabar-Matriks
## Penjelasan singkat program
Program ini dibuat menggunakan konsep modul Python untuk melakukan beberapa operasi matriks, yaitu penjumlahan matriks, perkalian matriks, dan perhitungan determinan matriks 3×3. Fungsi-fungsi tersebut disimpan pada modul keisha044.py, kemudian dipanggil pada program utama menggunakan perintah import. Dengan menggunakan modul, kode menjadi lebih terstruktur, mudah digunakan kembali, dan sesuai dengan ketentuan tugas yang mengharuskan adanya minimal satu fungsi dalam modul yang dapat berjalan dengan baik.
